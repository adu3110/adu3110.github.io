Oct. 2018

Project Link

Reference: Benner, P., Faßbender, H. On the Solution of the Rational Matrix Equation. Benner, Faßbender (2007)

BAYESIAN BINARY PROBIT

Oct. 2018

Project Link

This self-contained package implements Bayesian Auxiliary Variable Model for Binary Regression; performs Bayesian estimation of coefficients of Binary Logistic Regression with Probit link. The codes are written in C, which achieves 40x improvement in speed
Reference : Holems, C., Held, L. Bayesian Auxiliary Variable Models for Binary and Multinomial Regression
